# Casinito PRO - Nivel 4

Archivos incluidos:
- casinito-pro.html (juego principal)
- spin.wav, win.wav, lose.wav (sonidos de ejemplo)

Cómo usar:
1. Extrae el ZIP.
2. Abre `casinito-pro.html` en tu navegador (doble click).
3. Si el audio no arranca, hacé click en un botón para activar el AudioContext.
4. Subí tus propios sonidos desde la interfaz si querés reemplazarlos.
5. Para publicar: subí los archivos al repositorio y activá GitHub Pages.

Nota: los sonidos incluidos son tonos simples de ejemplo. Puedes reemplazarlos por MP3/OGG.
